   Predefined global object 

  window 

   #window is the root object for BOM(Browser object model)
   #window object represents browser window ,and is used to manipulate browser window

  objects and functions available through window
    1.document :- represents html document in browser
    2.closed:-returns true when window is closed
    3.console:-represents console of a browser
    4.frames:-returns all window objects running in the window
    5.history :- returns history object for window

                 history.go(number)
                 history.back()
                 history.forward()
    6.innerHeight:-returns height of window content area 
    7.innerWidth:-returns width of window content area
    8.Navigator 
            #Geolocation object in navigator is used to retreive latitude and longitude 
  coordinates of a device
